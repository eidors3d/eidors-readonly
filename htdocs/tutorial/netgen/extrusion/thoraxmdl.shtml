Modeling EIT current flow in a human thorax model

This example shows how EIDORS can by use Netgen to model the body shape of a human and then use it to build a reconstruction algorithm and see the current flow in the body. For this exmample, you need at least Netgen version 4.9.13.

Load thorax shape and identify contours

This image is from a human CT of a healthy man, provided with permission. Hand registered contours are available in this file [CT2.mat].
