Older EIDORS tutorials

These are older tutorials for EIDORS. They still work (we try hard to keep backwards compatibility), but they are no longer the recommended way to write EIDORS software (we've come up with cleaner functions to do the same thing).
To run these tutorials, you need to download and install EIDORS and then run this command in a matlab (or octave) session. 
>>run /path/to/eidors3d/startup.m
