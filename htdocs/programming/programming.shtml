EIDORS: Information for developers

Getting the EIDORS Source

  • Windows:
    − Download and install Tortoise CVS
    − Right click in the directory you would like to download to, and type in the value of CVSROOT below.
    − In the box Module enter eidors3d
  • Unix / Linux: Type
    export CVS_RSH=ssh
    cvs -d$CVSROOT checkout eidors3d
Use the following values for CVSROOT
  • Anonymous Access: Available to everyone
    CVSROOT=:pserver:anonymous@eidors3d.cvs.sf.net:/cvsroot/eidors3d
  • Developer Access: Available to registered eidors3d developers
    CVSROOT=:ext:DEVELOPERNAME@eidors3d.cvs.sf.net:/cvsroot/eidors3d

EIDORS Mailing List

  • Send message to the mailing list at EIDORS3D@listserv.umist.ac.uk
  • To subscribe, send mail to EIDORS3D-request@LISTSERV.UMIST.AC.UK with the body text SUBSCRIBE EIDORS3D your name
  • Mailing list archive is: http://listserv.umist.ac.uk/archives/eidors3d.html

Becomming an EIDORS Developer

  1. Go the the EIDORS project page
  2. Click on the link for New User via SSL. Register your username
  3. Send a request for membership to the maintainers
