Project Goal

GREIT is a collaborative project to develop a consensus "best-of-breed" linear reconstucution algorithm for lung EIT.

Getting Started

GREIT is in active development. No software has yet been released.

  • Browse Developer Versions
    Browse CVS

  • Read only access to Developer Version
    Read-only, worldwide access is available via CVS using Anonymous CVS Access with 
    cvs -d:pserver:anonymous@eidors3d.cvs.sf.net:/cvsroot/eidors3d co GREIT

  • Access Developer Version
    Read-write, access is available via CVS for registered developers. To become a developer, email the maintainers. Access CVS via: 
    cvs -d:ext:developername@eidors3d.cvs.sf.net:/cvsroot/eidors3d co GREIT
