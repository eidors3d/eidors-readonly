RemoveJacob

PURPOSE

RemoveJacob Removes the derivatives corresponding to the current carrying electrodes

SYNOPSIS

function Jh=RemoveJacob(J,L);

DESCRIPTION 

RemoveJacob Removes the derivatives corresponding to the current  carrying electrodes
 Function Jh=RemoveJacob(J,L);

 current carrying electrodes.
 Change the voltages accordingly, see RemoveVolt.m.

 INPUT

 J = Jacobian
