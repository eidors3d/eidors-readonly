elem_dim

PURPOSE

ELEM_DIM: dimension of elements in space (are elements in 2D or 3D space)

DESCRIPTION 

 ELEM_DIM: dimension of elements in space (are elements in 2D or 3D space)
  Note, elem_dim is not the same as the model dimention (can have 2D models in 3D space)
