Project Goal

The idea of this project is to promote a collaboration between groups working on Electrical Impedance Tomography (EIT) and Diffusion based Optical Tomography, in medical and industrial settings. The aim is to produce a suite of programs which perform mesh generation reconstruction and display for both techniques. We hope that we can produce robust, reliable and fairly portable software which draws on our collective expertise and implements some of the latest innovations.
