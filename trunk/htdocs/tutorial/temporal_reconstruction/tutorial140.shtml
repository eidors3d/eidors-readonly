Image reconstruction of moving objects

One advantage of EIT is the relatively high data frame rates. It is possible to take this into account using a reconstruction algorithm which accounts for inter frame correlations. The classic algorithm for this is the Kalman filter.

Figure: Image reconstructions of a moving ball in a medium. Top: using a classic one-stop solver, Bottom: using a Kalman solver. Note the warm up period of the Kalman images when the time correlations are being trained.
EIDORS can animate the resulting reconstructed images, as follows 
animate_reconstructions('tutorial140b', imgs1(1:20));
animate_reconstructions('tutorial140c', imgs2(1:20));
Output images are:
−Gauss-Newton solver
−Kalman solver
