GREIT Software

These tutorials show how to implement and test GREIT software with EIDORS (Note that EIDORS is required for all GREIT implementations) To run these tutorials, you need to download and install EIDORS and then run this command in a matlab (or octave) session. 
>>run /path/to/eidors3d/startup.m
