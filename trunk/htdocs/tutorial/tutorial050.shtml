More sophisticated use of Netgen

In order to make a more sophisticated use of netgen, it is possible to create models with inclusions. One sophisticated example is the function move_the_ball. This data simulates a moving ball in a 3D tank. You may either run 
>>move_the_ball(16,1)
or load the data directly from a file in the sample_data directory. 
>>load netgen_moving_ball
This will simulate data based on a ball shaped inclusion in a cylindrical tank.
