EIDORS fwd_models

Basic EIDORS data structures: the fwd_model

To understand a the basic EIDORS data structures, we define an electrical resistor, with ground at one end, and a single electrode at the other.
Output image: this shows a single line representing the resistor in 3D. Not very interesting, but this is a simple example.
In order to simulate the voltages, define an electrical resistor, with ground at one end, and a single electrode at the other. 


>> disp(data_1k)
    name: 'resistor model data'
    meas: [3x1 double]
    type: 'data'

>> disp(data_1k.meas')
    1.0200    2.0400    3.0600
