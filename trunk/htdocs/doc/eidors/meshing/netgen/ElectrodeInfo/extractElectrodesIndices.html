SYNOPSIS

function elec=extractElectrodesIndices(number_of_electrodes,surf)

DESCRIPTION 

 elec = extractElectrodesIndices(number_of_electrodes,surf)
 
 Used with readngneutral.m file and extracts the electrodes 
 elements from the surf matrix
 number_of_electrodes: Number of electrodes
 surf: Matrix generated by readngneutral.m containing surface elements
 elec: returns a [number_of_electrodes X 6] matrix that contains the electrode elements
